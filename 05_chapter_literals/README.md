## Key Takeaways

- JavaScript numbers are 64-bit floating point internally.
- You can write the same integer in decimal, binary, octal, or hexadecimal.
- `null` is an intentional absence of value; `undefined` means a variable has been declared but not assigned.
- Template literals (backticks) allow multi-line strings and interpolation: `` `Hello ${name}` ``.
