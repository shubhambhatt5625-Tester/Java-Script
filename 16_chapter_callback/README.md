## Key Takeaways

- A callback is a function passed as an argument to another function.
- Callback hell occurs when multiple nested callbacks make code hard to read and maintain.
- Synchronous callbacks execute immediately; asynchronous callbacks execute later.
- Playwright extensively uses callbacks for browser/page interactions.
- Promises and `async/await` were introduced to solve callback hell.
