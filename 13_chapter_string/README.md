## Key Takeaways

- Strings are immutable; methods return new strings.
- `slice(start, end)` is the most versatile extraction method.
- `includes()` returns boolean; `indexOf()` returns position (-1 if not found).
- `replace()` replaces only the first match by default; use regex with `/g` for global replacement.
- Template literals (backticks) simplify string interpolation and multi-line strings.
