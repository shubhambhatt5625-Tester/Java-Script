## Key Takeaways

- Function declarations are hoisted; function expressions are not.
- Arrow functions have shorter syntax and lexically bind `this`.
- IIFE runs immediately and creates its own scope.
- Rest parameters collect arguments into an array; spread expands iterables.
- Closures allow inner functions to access outer function variables even after the outer function returns.
- Higher-order functions take functions as arguments or return them.
