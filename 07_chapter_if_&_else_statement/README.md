# Chapter 07 - If & Else Statements

This chapter covers conditional logic using `if`, `else if`, and `else`. Each file is a practical exercise or real-world scenario.

## Files

| File | Description |
|------|-------------|
| `48_voting_rights.js` | Check if a person is eligible to vote based on age |
| `49_if_elseif_else_percentage.js` | Grade classification based on percentage |
| `50_real_example_if_else_login.js` | Simple login validation simulation |
| `51_if_else_of_api_status_code.js` | Handling HTTP status codes with conditionals |
| `52_IQ_if_else.js` | IQ / puzzle question using conditionals |
| `53_if_else_username_password.js` | Username and password validation |
| `54_IQ.js` | Another IQ-style conditional exercise |
| `55_odd_even.js` | Determine if a number is odd or even |
| `57_Grade_calculator.js` | Comprehensive grade calculator |
| `58_leap_year.js` | Check if a given year is a leap year |

## Key Takeaways

- `if` checks a condition and executes a block if true.
- `else if` allows chaining multiple conditions.
- `else` catches all remaining cases.
- Real-world validations (login, API codes, grades) are built using these blocks.
