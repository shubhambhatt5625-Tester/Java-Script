## Key Takeaways

- `switch` compares a value against multiple `case` labels.
- Without `break`, execution falls through to the next case.
- `default` acts like `else` — it runs when no cases match.
- Useful for clean, readable multi-branch logic (e.g., menus, enums, status codes).
