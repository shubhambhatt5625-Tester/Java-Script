## Key Takeaways

- Arrays are zero-indexed ordered collections.
- `push`/`pop` work at the end; `unshift`/`shift` work at the beginning.
- `splice()` can add, remove, or replace elements at any position.
- `map`, `filter`, and `reduce` are essential for functional array operations.
- `slice()` returns a shallow copy; it does not modify the original array.
