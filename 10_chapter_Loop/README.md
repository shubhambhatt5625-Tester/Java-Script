## Key Takeaways

- `for` loop is best when the number of iterations is known.
- `while` loop runs as long as the condition is true.
- `do-while` guarantees at least one execution before checking the condition.
- `break` exits the loop entirely; `continue` skips to the next iteration.
- Nested loops and loop + `if-else` combinations solve complex pattern problems.
