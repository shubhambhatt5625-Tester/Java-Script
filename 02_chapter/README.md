# Chapter 02 - Reserved Keywords

This chapter explains why JavaScript reserved keywords cannot be used as variable names or identifiers.

## Files

| File | Description |
|------|-------------|
| `04_keyword_cannot_be_name_as_variable_name_or_identifier.js` | Demonstration of keyword restrictions |

## Key Takeaways

- Words like `let`, `var`, `function`, `if`, `else`, `return`, etc. are reserved by the language.
- Using them as identifiers causes a syntax error.
